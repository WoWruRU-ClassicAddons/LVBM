
Howto Install La Vendetta Boss Mods?

Simply extract the zip to the folder \Interface\Addons to get the following folders:

\Interface\Addons\LVBM_API
\Interface\Addons\LVBM_GUI
\Interface\Addons\LVBM_RaidTools
\Interface\Addons\LVBM_NAXX
\Interface\Addons\LVBM_Other
\Interface\Addons\LVBM_AQ20
\Interface\Addons\LVBM_AQ40
...

if you update from an older Version, please delete the old

\Interface\Addons\La_Vendetta_Boss_Mods
